## latex-template
Шаблон для отчетов и РПЗ МГТУ

## Environment
- Операционная система: Linux, Windows;
- Компилятор: pdflatex.

## Dependencies
- texlive. Установка через ```sudo apt install texlive```;
- LaTEX-класс bmstu. Установка через ```sudo tlmgr install bmstu```.

## Build and run
cd main-service
mkdir build
cd build
cmake ..
